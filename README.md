# Riemann Surface Notes

Notes on MATH5306: Riemann Surfaces, written in [Typst](https://typst.app/).

## Compiling

To compile the notes into a PDF locally, you will need to install the Typst CLI. Then, simply run the following command in the repository root:

```sh
typst compile main.typ
```

This will generate a `main.pdf` file containing the compiled notes.

## References

1. O. Forster, *Lectures on Riemann Surfaces*
2. Griffiths & Harris, *Principles of algebraic geometry*
